## use

- step 1 clone this repo

```
git clone https://github.com/tahsinzidane/backend.git
```

- step 2 navigate the folder

```
cd 'C:\Users\path\to\the\backend>'
```
- step 3 install node modules
```
npm i 
```

- step 4 & final last step run the server
```
npm run dev
```
